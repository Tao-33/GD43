# Program：CIMC_GD32_KEY_driver

## 程序简介
- 工程名称：GD32F470 KEY驱动
- 实验平台: CIMC IHD V0.4
- MDK版本：5.25
CIMC_GD32_KEY_driver

## 板载资源

 - GD32F470VET6 MCU
 - LED * 4
 - KEY * 4
 
## 功能简介

程序模板，可以用来拷贝建立工程
利用GD32F470VET6单片机实现用按键操作LED点亮功能。

## 实验操作
下载程序后复位开发板，按下不同的KEY，对应的LED点亮

## 引脚分配

		KEY1   <--->     PE2
		KEY2   <--->     PE3
		KEY3   <--->     PE4
		KEY4   <--->     PE5

		LED1   <--->     PA4
		LED2   <--->     PA5
		LED3   <--->     PA6
		LED4   <--->     PA7
